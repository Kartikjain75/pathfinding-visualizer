🧭 Pathfinding Visualizer

An interactive Pathfinding Algorithm Visualizer built in Python (Pygame) that demonstrates how different algorithms like A*, Dijkstra’s, BFS, and DFS find optimal paths in a grid.

🚀 Features

  • Visualize A*, Dijkstra’s, BFS, and DFS algorithms in real-time
  • Adjustable grid size and speed
  • Interactive start and end node placement
  • Path and visited node animations

⚙️ Installation & Setup

git clone https://github.com/Kartikjain75/pathfinding-visualizer.git
cd pathfinding-visualizer
pip install pygame
python advanced_pathfinding_visualizer.py
